The Homepage

Purpose
Allows initial access to the available web pages in the Plant 19 Website.
 

To Access Individual Pages
  • Place cursor over the desired web page menu item.
  • Both the Kiosk Database Manager and MTMS Location Maintenance menu items expand to allow access to the “Add New Entry” and the “View/Perform Maintenance” pages. To access one of these pages, place cursor over the desired menu item and click on the item.
  • The Quality Control Lot Traceability page is a “Search” only page. To access this page, place cursor over it’s menu item and click on the item.
