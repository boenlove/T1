Kiosk Database Manager: View / Perform Maintenance

Purpose
This page will allow you to view, update and delete Standard Pack's for a specific Part Number/Package Code record.
 

About this page
  • All fields are required, therefore if one or more fields are left blank, the field will be marked with a red asterisk and a message box will pop up indicating the errors that have occurred. The same will happen for those fields which require numeric entry.
  • NOTE: This is a secure page, a security check will be done based on the user ID of the individual attempting to edit or delete a record. If the user does not have security to to make changes to the information, the edit/delete options will be disabled.
  • If you do not have security and believe that you should, please contact the Data Owner

Viewing a Standard Pack
  1. Select a Plant
  2. Select a Department
  3. Select a Part Number/Package Code
  4. If Standard Packs exist for the above combination, a list of Standard Packs will display

Updating a Standard Pack
  1. Select a Plant
  2. Select a Department
  3. Select a Part Number/Package Code
  4. If Standard Packs exist for the above combination, a list of Standard Packs will display
  5. Click the Edit button next to the Standard Pack you wish to Edit
  6. Make the necessary changes and click "UPDATE"

Deleting a Standard Pack
  1. Select a Plant
  2. Select a Department
  3. Select a Part Number/Package Code
  4. If Standard Packs exist for the above combination, a list of Standard Packs will display
  5. Click the Delete button next to the Standard Pack you wish to Delete

Adding a Standard Pack
This function is performed on a different screen. Click on the "ADD" button to be taken to the Add Standard Pack screen.
