MTMS Maintenance: View / Edit Location

Purpose
This page will allow you to  view, update, and delete MTMS Location records for a particular Part Number/Package Code record.
 

About this page
  • NOTE: This is a secure page, a security check will be done based on the user ID of the individual attempting to edit or delete a record. If the user does not have security to to make changes to the information, the edit/delete options will be disabled.
  • If you do not have security and believe that you should, please contact the Data Owner

Viewing MTMS Locations
  • To view all MTMS Location records, leave the Part Number and Location fields blank, then click on the View/Edit button.
  • To view the location record based on Location only, leave the Part Number field blank and select a location.
  • To view a MTMS location record based on Part Number and Location, select the part number. Once a part number has been selected, its corresponding Location will automatically be selected and the record will be displayed.

Updating a MTMS Location
  1. Select a Part Number
    • The Location for the selected part number will automatically be selected and the Location record will display.
  2. Click the Edit button next to the MTMS Location you wish to Edit
  3. Make the necessary changes and click "UPDATE"

Deleting a MTMS Location
  1. Select a Part Number
    • The Location for the selected part number will automatically be selected and the Location record will display.
  2. Click the Delete button next to the MTMS Location you wish to Delete

Adding a new MTMS Location
This function is performed on a different screen. Click on the "ADD" button to be taken to the Add MTMS Location screen.
