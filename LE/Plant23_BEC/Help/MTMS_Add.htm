MTMS Location Maintenance: Entry Form

Purpose
To assign a part number to an MTMS location.
 

About this page
  • If you do not have security and believe that you should, please contact the Data Owner

Assigning a Part Number to an MTMS Location
  1. Enter a valid Part Number
  2. Enter a Location name
  3. Click on Add Location

NOTE: If the part number is not valid, an error will display.
