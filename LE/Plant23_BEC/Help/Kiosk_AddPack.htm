Kiosk Database Manager: Add Standard Pack

Purpose
To add Standard Pack information to a particular Part Number/ECL.
 

About this page
  • All fields are required, therefore if one or more fields are left blank, the field will be marked with a red asterisk and a message box will pop up indicating the errors that have occurred. The same will happen for those fields which require numeric entry.
  • NOTE: This is a secure page, therefore after a Plant is selected from the list, a security check will be done based on the user ID of the individual attempting to add a new entry. If the user has security to add, then the Department drop down list with all available departments within the Plant, will automatically be populated.
  • If you do not have security and believe that you should, please contact the Data Owner

Adding a Standard Pack
  1. Select a Plant
  2. Select a Department
  3. Enter the appropriate information in the rest of the fields
  4. Click the "ADD" button

Editing a Standard Pack
This function is performed on a different screen. Click on the "EDIT" button to be taken to the Edit Standard Pack screen.
